闲言碎语-2019

2020-01-01

非常认同，周星驰的电影从 2000 年往后的都不好看，包含但不限于《少林足球》、《长江七号》、《功夫》等等，唯一的亮点算是塑造了一个包租婆的银幕形象，实在是找不出其他亮点。当然这部电影《新喜剧之王》我肯定不会去电影院看的。

社会娘炮化越来越严重了：迎男而上，强人锁男，男上加男，难为其男，知难而上，进退两男，男解男分，面有男色，心痒男忍，左右为男，换男见真情，好梦男成。

USB Type-C 这个东西我是越来越喜欢了，希望将来的接口都是这个标准接口。这会节省很多数据线资源。

一个农历新年就清楚了抖音的用户群体，村镇用户真心不少。希望抖音慢慢发展，完成农村包围城市，在国内能有一片天地。

手机里还有十几年没联系过的手机号，不知道该不该删除。

最终我还是选择了随波逐流，刚刚买了停车位15.8万全款，三分之二的钱是从信用卡里刷出来的，现在我也加入了卡奴的大军了。

银联云闪付支持工行信用卡账单查询了，以后查信用卡账单一个云闪付就够了。云闪付手握这么好的资源却办不成啥事。。。

island 支持恢复appops设置了，千呼万唤的功能终于出来了，现在还差island内外文件分享的功能，这个做出来之后就更好了。

今天总感觉心情比较舒畅，不知道咋回事，明明ETF赔钱了。

315 关于的银联闪付盗刷问题，其实压根不用担心什么，毕竟银联POS机这块可控性非常强，并且还是非实时清算，一旦发现盗刷可以马上阻止这笔记交易进行清算，通常拥有 POS 机的用户都算是有点资质的用户，不会因为一点小钱而走险去盗刷的。

两年半没骑行了，今天把车子气打满，出去骑行了一个多小时，感觉很不错，找回了当年的感觉。

从2012年开始追权力的游戏，今天终于结束了，这个应该是我追过的最长的一部电视剧。感慨还是有的，只是很多坑没填呢，希望能出一些番外篇把坑填满。

最近几个月都是早上五点左右醒来，晚上十一点以后睡觉，中午也不午休，我也比较奇怪我怎么扛得住。必须得做出改变了，先尝试早点睡觉，如果不可行， 那么就需要找医生看看了，再这样下去我觉得我会扛不住的。

今天参加一个同学的婚礼，从西安到泰安，第一次这么远的来参加婚礼，有点折腾，但是见到了好几个多年未见的同学，一起打勾级，玩到了凌晨，又和另一同学夜聊了两个小时。虽然两天时间比较紧张，也比较累，但是很充实。

同事关系本来就是那样，是我自己看的太重了。有些时候我不善于表达，但是很多事情我自认为做的到位。

最近几天一直在纠结 YAML 的格式，查了不少文档，最终感觉还是用下面的方式比较好些。

spec:
- name: zhangsan
  age: 15
- name: lisi
  age: 16
status:
  running: true

可能是平时工作强度太大了吧，周末一点都不想看工作相关的东西，一点都不想看。但是周末时间又需要消耗掉，以前是追美剧，最近一年也没再追了，感觉没意思。这周一直在折腾翻墙的东西，深入了解了一下 shadowsocks plugin 和 v2ray ，以及他们的客户端。最后的结论是翻墙主要靠的还是靠国内连国外服务器的速度。

刚刚把招商银行的掌上生活应用给卸载了，上个月卸载了中信银行的动卡空间应用，以后尽量不用这两家银行了，商业银行套路太多，防不胜防。回归三大行（工行、中行、建行），无套路、无优惠，刷卡简单粗暴，这挺好，省得浪费时间和精力在所谓的“羊毛”上面。

今晚带娃出去玩，去肯德基吃了一顿，喝了杯咖啡，然后失眠了，每次喝咖啡都失眠，以后坚决不喝了。

很生动的一张图，富人越来越富，穷人越来越穷，将来希望能有种方式打破这个怪圈。“三十年河东，三十年河西”在现在这个社会变成了“三十年河东，三十年仍然是河东”，穷人很难出头了。（图片省略）

刚刚体验了一把支付宝的刷脸支付，确实牛逼，以后不怕手机没电了。

阅文集团是我买的第一支股票，也是第一次打新，可惜后来一直在跌，给我的教训不小，一共赔了一万五，现在全仓小米。

国内证券市场竞争确实太激烈了，现在有些证券厂商基本免费了，万分之0.5收费场内基金，还免双5，这个挺好，弄的我都不想支付宝定投了。

今天整理了一下，我发现的投资渠道有十数个了，感觉有点多，但是各个渠道又有各个渠道的用处，不想全放在一起。管理起来确实有点麻烦，钱是过不去的坎啊，身边比较有钱的人都投资房子了，像我这样资本不多的，只能找其他方式投资了。

回归信用卡的本质上来，注重用卡，而非玩卡。

把招商银行的银联实体信用卡注销，换成了银联电子卡，因为我以后不准备用招商银行了，留个实体卡还要关心年费的事情，麻烦。

时间过的挺快，娃刚学会走路时我心里还想：娃啥时能跑的很稳啊。而现在娃走路和跑路已经比较平稳了，摔倒的次数明显减少了很多。

昨天升级系统，结果丢失了数据，无奈最后重新刷了一下系统，所有东西从头来过。手机这东西还是买稳定可靠的好，下部手机大概率会是小米或 Google 的。

真是没有对比就没有伤害，之前一直觉得平安证券app不好用，用了海通证券app之后我才发现平安证券是真心好用啊。

准备把摩拜单车应用卸载了，用美团算了，美团功能还更多一些。以后美团也是手机里的必装应用之一了，应用越来越多了。

上周有三天都是骑电动车上下班，城市道路一直都在建设或修修补补，非机动车道基本是废的，被各种机动车、非机动车占道，或者道路中间有大坑过不去，总之想在非机动车道上骑行实在太难了。然后就有很多非机动车在机动车道上行驶，导致机动车不好行驶。总之就是各种不爽。

拥抱 Podman，我在自己的 VPS 上面已经把 Docker 卸载并安装上了 Podman，终于可以不用启动 Docker 的守护进程了，Docker 太笨重了，问题太多了。

百度地图越来越烂了，最近几次导航都不行。一直用百度地图，收藏了很多地点，也没法导出来。准备用一段时间高德地图时时，希望高德地图给力一点。

Podman 有个很重要的功能就是可以编排 Kubernetes Pod，这让我感觉很舒服，我自己的翻墙工具和网站就是由多个容器组成的，用 Pod 来启动是最好不过的了。

突然怀念十年前的 Web 2.0 时代了，很多牛人写独立博客，内容很具有阅读性，学习价值非常大，算是无私奉献吧。而现在很难获取到有价值的信息了，到处都是封闭，各种知识付费。

最近两天在 YouTube 上订阅了很多频道，以后打发无聊时间的好去处。

都忘记从啥时开始就不追美剧了，以前漫威和 DC 的好几个超级英雄电视剧我一直再追，后来越来越感觉没意思了，就没再追，现在压根不看美剧了，看的都是电影解说，快餐消费，直接看别人消化完的内容。

折腾了很久，终于把 Lofter 的内容迁移到了 Tumblr ，虽然不是完全迁移，但是大部分内容都迁移了，只是丢了一些图片，以后再也不敢把内容放在别的平台上了，太不靠谱了。

自己打脸了，我昨天没忍住办了个银联钻石和 Visa infinite 卡，原因是我还没这么高级别的卡，需要每个月刷 4 笔交易来免年费，以后有的折腾了。

公交车免费确实很方便，小县城也许只能给出这样的惠民政策了吧。

买的联通校园卡没法在老家用，因为老家没有联通4G信号，不得不又买了个移动流量卡，希望能用吧。

微信的恶意账号检测机制真垃圾，给家人注册俩号都被封了，现在没手机号了，家人手机用不了微信。

自从 Pixel 4 出来之后我对Google的信仰已经坍塌了，不对他抱有希望了。我喜欢用的是 Google 全家桶 + 小米钱包，现在已经不管是不是原生了，只要这俩东西能让我用的爽，并且续航可以，我就无所谓了。MIUI 在某种程度上说已经非常棒了。

刚刚删除了很多电话号码，有初中、高中和大学同学的，有工作后认识的人的，这些都是五年以上没联系过的人。我是个怀旧的人，一直不舍得删除这些人的联系方式，但是今天突然想清理一下通讯录了。

Redmi K20 Pro 模拟小区门禁卡已经成功，使用国内手机和系统的一个好处就是各种实用功能，并且还不用太折腾。

MIUI11+Google全家桶，体验其实还好，常用的那几个应用还是一样，基本没啥区别，防骚扰能力提升了一大截，应用停止状态也能收到消息提醒。

越来越喜欢小米手机和MIUI11了，以后我估计可能只会买小米手机了。

了却了一桩心事，媳妇的移动手机号终于转成联通了，下个月加到宽带家庭套餐里。

交大口腔医院（三甲）只有治病没有服务，瑞泰尔仓口腔医院的服务真心不错，只是没法报销，否则我就去他们这里治疗了。

自从用了 MIUI11 之后就一直在纠结非圆形图标的问题，已经不纠结了，反正我也不用 MIUI 自带的 Launcher ，什么图标都无所谓啦。

今天把博客转到了 Hugo 并且开启了 Dark Mode ，整体还挺满意的。从 Octopress 到 Hexo 再到 Hugo，一路折腾，这次之后估计不能再折腾了。

支付宝绝对是一大良心应用，这次更新去掉了存储权限，手机存储里干净了很多。

原本以为小米公交卡吸引力很大呢，等我用了小米手机之后才发现限制太多，比如：无法移卡或需要主动移卡，这个在手机丢失或坏掉的情况下公交卡就作废了，这个是我最不能接受的，当然这不是小米能决定的，毕竟这是公交公司的要求。现在京东和支付宝也出NFC交通卡了，目前是下发到华为钱包里，这个不需要预付费，用多少充多少，也许这个是趋势。京东目前是以押金的形式来做到这点的，阿里因为有花呗，所以我估计阿里会以花呗冻结额度这种方式来做到免预存。

本来想买个乐歌E3自动升降书桌，可我现在改变了主意，想自己找人给定制一个合金书桌架，然后再买个书桌面板。

最终买了个手摇升降式办公桌，大小刚合适，有两点不足：桌面材质不行厚度也不像卖家说的那么厚，升降到高点桌子不太稳。
